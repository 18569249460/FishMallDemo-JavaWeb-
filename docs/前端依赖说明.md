# 前端依赖说明

`frontend/package.json` 是标准 JSON 文件，语法不允许写注释，所以前端依赖用途单独记录在这里。

## dependencies

- `vue`：Vue 3 核心框架，用于构建用户端和管理端页面。
- `vue-router`：前端路由，用于首页、商品、购物车、个人中心和后台页面之间切换。
- `pinia`：状态管理，用于保存登录 Token、用户信息等全局状态。
- `axios`：HTTP 请求工具，用于调用后端 `/api/**` REST 接口。
- `element-plus`：Vue 3 UI 组件库，用于表单、表格、按钮、弹窗、菜单等后台和前台基础组件。
- `@element-plus/icons-vue`：Element Plus 图标库，后续补按钮图标和操作入口时使用。

## devDependencies

- `vite`：前端开发服务器和构建工具。
- `@vitejs/plugin-vue`：让 Vite 支持解析 Vue 单文件组件。

## scripts

- `npm run dev`：启动前端开发服务器，默认端口 `5173`。
- `npm run build`：打包前端生产文件，输出到 `frontend/dist`。
- `npm run preview`：本地预览生产构建结果。
